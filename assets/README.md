# Assets

This folder contains source and reference graphics used for documentation and
future figure development.

- `binsfeld_2025_fig1_screen_overview_reference.png` is a reference copy of
  Fig. 1 from Binsfeld et al. (2025), PLOS Biology,
  <https://doi.org/10.1371/journal.pbio.3003260>. It is retained as a visual
  reference for a future DStressR pipeline figure and should be redrawn or
  adapted with proper attribution before use in a manuscript figure.
